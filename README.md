# .github

This repository contains default community health files for the
[lakersjunior19-star](https://github.com/lakersjunior19-star) GitHub organization.

Files placed here (such as `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, and issue/PR templates)
are automatically used by all repositories in the organization that do not define their own.
